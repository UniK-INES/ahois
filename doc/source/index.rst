
AHOIS-pro documentation
***********************

The AHOIS-pro documentation is structured relating to its target groups:

 * :ref:`Stakeholder <stakeholder>` reports the vision, important feature and focuses on input and output.
 
 * :ref:`Application <application>` is for users of the model and provides an ODD description, describes relevant concepts,
   offers recipes and the API.
   
 * :ref:`Development <development>` provides information related to the code repository, 	a style guide and further information
   about documenting, testing, logging, and localisation.
